Quick Start Example

Generate a new FastAPI route with the create-fastapi-route blueprint:

curl -X 'POST' \
  'http://localhost:8000/api/blueprints/generate' \
  -H 'Content-Type: application/json' \
  -d '{
  "blueprintId": "create-fastapi-route",
  "parameters": {
    "routeName": "products",
    "modelName": "Product",
    "outputPath": "./output/products.py",
    "routePrefix": "/api",
    "tags": ["products"]
  }
}'
